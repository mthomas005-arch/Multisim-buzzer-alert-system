# Multisim-buzzer-alert-system
Designed and simulated a logic-based buzzer alert system using Multisim, implementing AND gates, D flip-flops, LEDs, and a buzzer to demonstrate digital control and alert mechanisms.

 Project Overview

The system processes multiple digital inputs through AND gates and D flip-flops to control LEDs and activate a buzzer when predefined logic conditions are met.  
It simulates how digital logic can be used in alarm systems, access control, and alert mechanisms.

---

 Key Components Used

- 7408 AND gates
- CD4013 D flip-flops
- OR gate
- LEDs for visual indication
- Buzzer for audible alert
- Push-button inputs
- Resistors and 5V DC power supply

---

 System Operation

- User inputs are provided using push buttons
- Logic conditions are evaluated using AND gates
- Flip-flops store and control system states
- LEDs indicate logic states
- A buzzer is triggered when the output logic condition is satisfied
